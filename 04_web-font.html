개발자 서은정

클라우드 엔지니어를 꿈꾸는 개발자

끊임없이 학습하고 성장하는 개발자, 클라우드 기술로 세상을 연결합니다

기술 스택

⚡

JavaScript

동적이고 인터랙티브한 웹 애플리케이션을 구현하며, ES6+ 문법과 비동기 프로그래밍에 능숙합니다.

🗄️

MySQL

효율적인 데이터베이스 설계와 최적화된 쿼리 작성을 통해 안정적인 데이터 관리 시스템을 구축합니다.

🎨

HTML/CSS

시맨틱한 마크업과 모던 CSS 기법을 활용하여 사용자 친화적이고 반응형 웹 인터페이스를 제작합니다.

서은정의 이야기

안녕하세요! 저는 클라우드 엔지니어를 꿈꾸는 개발자 서은정입니다. 기술을 통해 사람들의 삶을 더 편리하고 풍요롭게 만드는 것이 저의 목표입니다.

현재 JavaScript, MySQL, HTML/CSS를 기반으로 웹 개발 실력을 쌓아가고 있으며, 앞으로 AWS, Docker, Kubernetes 등의 클라우드 기술을 마스터하여 확장 가능하고 안정적인 시스템을 구축하는 엔지니어가 되는 것이 목표입니다.

새로운 기술을 배우는 것을 즐기며, 문제를 창의적으로 해결하고 팀과 함께 성장하는 개발자가 되기 위해 노력하고 있습니다.
